> An interactive dashboard showcasing [Syra AI](https://syraa.fun)'s trading intelligence capabilities for the Solana ecosystem.

## ✨ Features

| Panel | Description |
|-------|-------------|
| **📊 Market Pulse** | Real-time sentiment gauge, risk scores, trending narratives |
| **📰 News & Headlines** | AI-curated news feed with sentiment indicators from 50+ sources |
| **🐋 Whale Tracker** | Large on-chain transactions, smart money flow charts, DEX activity |
| **🔍 Token Analyzer** | Search any token for AI-powered analysis with score rings & indicators |
| **🔗 API Explorer** | Interactive x402 payment flow demo & full endpoint directory |

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/matheusbombasseiam/syra-ai-dashboard.git
cd syra-ai-dashboard

# Open directly in browser
open index.html

# Or use a local server
npx http-server . -p 8080
```

Then visit `http://localhost:8080`

## 📝 About This Contribution

This project was built as a community contribution for the Superteam bounty, under the **Integration + Tutorial** track. It demonstrates Syra AI's capabilities through an interactive dashboard and provides a tutorial for developers looking to integrate the Syra API into their workflows.

> ⚠️ All data displayed in the dashboard is **simulated** for demonstration purposes. For real-time intelligence, use the [Syra Agent](https://agent.syraa.fun) or [API Playground](https://playground.syraa.fun).
