🔗 x402 Payment Flow
How the Syra API works — trustless, pay-per-request
1
Send API Request
Make a standard HTTP request to any Syra endpoint.
GET https://api.syraa.fun/v2/news Content-Type: application/json Body: { "ticker": "SOL" }
2
Receive 402 Payment Required
Server responds with payment instructions — price, currency, and method.
// Response: HTTP 402 { "price": 0.01, "currency": "USD", "network": "solana", "paymentMethod": "x402" }
3
Submit Payment
Complete the micropayment on Solana (USDC/SOL). No account or API key needed.
// Pay via Solana wallet PAYMENT-SIGNATURE: 4zF9k...x402 PAYMENT-TOKEN: eyJhbG...proof
4
Receive Data ✓
Retry the original request with payment proof. The API returns your data.
// Response: HTTP 200 OK { "data": [ { "title": "Solana DEX Volume Hits ATH", "sentiment": "bullish", "source": "CoinDesk" } ] }
